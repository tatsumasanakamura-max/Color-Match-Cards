# カラーマッチカード

カラーマッチカードは、色・数字・効果を合わせてカードを出していく、ブラウザだけで遊べるオリジナルカードゲームです。公式名称、公式ロゴ、公式カードデザインは使わず、独自の名前と見た目で作っています。

## アプリ概要

- 2人、3人、4人対戦を選べる静的Webアプリです。あなた以外はCPUが操作します。
- `index.html` をリポジトリ直下に置いたまま、GitHub Pagesで公開できます。
- ビルドや依存パッケージのインストールは不要です。
- iPhone、iPad、PCで場と手札を見ながら遊べるレイアウトを目指しています。

## 遊び方

1. タイトル画面で対戦人数を選び、「ゲーム開始」を押します。
2. 自分の番になったら、場のカードと同じ色、同じ数字、同じ効果のカードを出します。
3. 色変えカードはいつでも出せます。
4. 出せるカードがない場合は「1枚引く」を押します。
5. 手札が0枚になったプレイヤーの勝ちです。

## 基本ルール

- 最初に各プレイヤーへ7枚ずつ配ります。
- 数字カードは同じ色または同じ数字なら出せます。
- Skip / Reverse はカード中央に英字で表示します。Skipは次の1人を飛ばし、Reverseは順番を反転します。2人対戦のReverseはSkip相当です。
- 「+2」を出された側は、重ね出しできなければ累積枚数を引きます。
- 「色変え+4」を出すと、次の色を選び、相手に4枚分のドローを渡します。

## ローカルルール

ゲーム開始前の「ルール設定」でON/OFFできます。ゲーム中は変更せず、現在の設定だけを小さく表示します。

- 同じ数字まとめ出し: 同じ数字カードを複数枚まとめて出せます。数字カードだけが対象です。
- +2重ね: +2に+2を重ねられます。
- 色変え+4重ね: 色変え+4に色変え+4を重ねられます。
- +2に色変え+4を重ねる: +2の累積中に色変え+4を重ねられます。
- 10点以上のカードでは上がれない: Skip / Reverse / +2 / 色変え / +4 を最後の1枚として出せません。数字カードで上がる必要があります。

重要な累積ルール:

- +2 → 色変え+4 は、設定ONなら許可します。
- 色変え+4 → +2 は禁止です。
- +2 → +2 → 色変え+4 は、設定ONなら許可します。
- +2 → 色変え+4 → +2 は禁止です。
- 色変え+4 → 色変え+4 は、色変え+4重ねONなら許可します。

## ルールプリセット

- 標準ルール: すべてOFF
- 家族ルール: 同じ数字まとめ出し、+2重ね、色変え+4重ねをON
- 激しめルール: すべてON

## 対戦人数

対戦人数は2人、3人、4人から選べます。2人対戦は「あなた + CPU1」、3人対戦は「あなた + CPU1 + CPU2」、4人対戦は「あなた + CPU1 + CPU2 + CPU3」です。

複数人対戦でも、+2 / 色変え+4 の累積ドローは全体で累積し、出せなかった現在のプレイヤーがまとめて引きます。合計得点が少ないプレイヤーが最終勝者です。同点の場合は同点優勝として表示します。

## 起動方法

ローカルで簡単に確認する場合:

```bash
python -m http.server 8000
```

ブラウザで以下を開きます。

```text
http://localhost:8000
```

直接 `index.html` を開いても動く構成ですが、ES Modulesを使っているため、ブラウザによってはローカルサーバーでの確認が安定します。

## GitHub Pages公開方法

1. GitHubのリポジトリ画面を開く

## 回戦数設定
ルール設定画面で、対戦形式を 1回戦 / 3回戦 / 5回戦 から選べます。初期値は 1回戦 です。

## 得点ルール
各回戦終了時に、全プレイヤーの残った手札を点数化します。全回戦終了後、合計得点が少ないプレイヤーが勝ちです。同点の場合は同点優勝です。

- 数字カード: 数字そのまま
- Reverse / Skip: 10点
- +2: 15点
- 色変え: 20点
- 色変え+4: 40点

手札が0枚になったプレイヤーのその回の点数は0点です。設定で、3回戦目・5回戦目の点数を2倍にできます。3回戦モードでは3回戦目だけ、5回戦モードでは3回戦目と5回戦目が2倍になります。1回戦モードではこの設定の影響はありません。

## 回戦途中の進め方
3回戦 / 5回戦では、各回戦終了後に今回の点数と合計点が表示されます。途中回戦では「次の回戦へ」を押すと、山札・捨て札・手札・現在色・累積ドローがリセットされ、合計得点とルール設定を維持したまま次の回戦を開始します。最終回戦後は最終得点と勝敗が表示されます。
